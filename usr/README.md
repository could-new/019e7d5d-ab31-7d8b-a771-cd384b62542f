# Digital RPP Ko-Kurikuler

Aplikasi Flutter ini merupakan representasi digital dari Rencana Pelaksanaan Pembelajaran (RPP) Ko-Kurikuler untuk kelas IV Fase B di Sekolah Luar Biasa (SLB) dengan tema "Peduli dan Berbagi di Sekolahku". Aplikasi ini didesain agar guru, orang tua, dan asesor dapat melihat modul ajar, aktivitas, instrumen asesmen, serta Lembar Kerja Peserta Didik (LKPD) yang berdiferensiasi secara interaktif di berbagai platform (Web, Mobile, Desktop).

## Fitur
- **Identitas & Profil**: Menampilkan data sekolah dan profil kesiapan belajar peserta didik (Raffa, Wildan, Suryadi).
- **Aktivitas Pembelajaran**: Jadwal aktivitas 10 minggu berbasis *Project Based Learning* (PjBL).
- **Asesmen**: Rubrik asesmen awal, proses, dan sumatif dalam bentuk tabel interaktif.
- **LKPD**: Lembar Kerja Peserta Didik yang berdiferensiasi sesuai tingkat kesiapan anak.
- **Responsif**: Tampilan menyesuaikan untuk layar handphone, tablet, dan desktop.

## Menjalankan Aplikasi
1. Pastikan Flutter SDK sudah terinstal.
2. Jalankan perintah `flutter pub get` untuk mengunduh dependensi.
3. Jalankan `flutter run` untuk memulai aplikasi di emulator atau web browser.
